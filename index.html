☁️ クラウド同期（GitHub Gist）

GitHub Gist をクラウドDBとして使い、管理者間でデータを共有します。GitHub・会社サーバーどちらでも動作します。
【初回設定手順】
① GitHub → Settings → Personal Access Token を発行（scopeはgistのみでOK）
② gist.github.com でプライベートGistを新規作成（ファイル名: aegis_sync.json、内容は {} でOK）
③ GistのURLの末尾ID（例: a1b2c3d4e5f6...）と発行したトークンを以下に入力して保存

✓ 保存しました

📣 以下の「同期コード」を社員に共有してください（Slack・メール等）
